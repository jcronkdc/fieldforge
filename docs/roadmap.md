# MythaTron Roadmap

## Horizon 1 · MVP Loop (Weeks 1–6)
- Swipe feed prototype with snackable story highlights.
- Instant “Remix” creation flow powered by narrative AI.
- Story genome tagging (mood, pacing, genre).
- Founding creator program + onboarding kit.
- LangBot-powered prompts across Discord/Slack/Telegram.

**KPIs:** swipe→remix conversion, creations per user, D7 retention, worlds with 3+ branches.

## Horizon 2 · Network Effects (Weeks 6–12)
- Persistent story worlds with canonical + fan branches.
- Immersive outputs (AI art, voice, story trailers).
- Creator analytics + monetization (tipping, premium arcs).
- Weekly “Story Arena” live events.
- Growth infrastructure (referrals, paid acquisition experiments).

**KPIs:** creator earnings, average worlds/user, event participation, new readers from shared trailers.

## Horizon 3 · Platform & IP Expansion (12+ Weeks)
- Open API & SDK for third-party integrations.
- Story governance tools, moderation, lore councils.
- Global localization & regional community programs.
- Partnerships for IP spin-offs.
- Advanced personalization & explainability dashboards.

**KPIs:** API integrations, international DAU share, creator retention, licensed IP deals, AI service reliability.

## Immediate Actions
1. Draft Phase 1 product spec (swipe feed, remix flow, story schema, KPIs).
2. Bootstrap ML workspace with tagging pipeline scaffolding.
3. Configure LangBot bots for daily cross-platform prompts.
4. Build founding creator outreach kit + incentives.
5. Define analytics events for swipe/remix funnel and retention tracking.
